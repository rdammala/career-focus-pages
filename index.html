Workflow Convention & Recreate Notes

Source of truth for regenerating any resume bundle

Exact workflow prompt

#### Step 1: Analyze the Job
- Identify key requirements, skills, and keywords from the job description
- Map my experience to each requirement

#### Step 2: Create These Deliverables

**A. Tailored Resume (1–2 pages max)**
- Professional Summary tailored to the role
- Technical Skills section aligned to JD keywords
- Experience section with bullets reframed to match role requirements
- Use quantifiable achievements wherever possible
- Include Education & Certifications

**B. Tailored Cover Letter (1 page)**
- Address specific role requirements
- Reference company mission/values
- Highlight 3–4 strongest alignment points
- Include location/shift availability if relevant
- Professional, confident tone

**C. Job Details Summary**
- Job link/URL
- Company name
- Position title
- Location
- Salary range (if available)
- Key requirements extracted
- Application status tracker

#### Step 3: File Organization

Create all files in this structure:
Resumes/2026/<CompanyName>/
├── Job_Details.md          (job link, company, full JD, status tracker)
├── Rajesh_Dammala_<Company>_<RoleShortName>.docx
├── Rajesh_Dammala_<Company>_<RoleShortName>.pdf
├── Rajesh_Dammala_CoverLetter_<Company>.docx
└── Rajesh_Dammala_CoverLetter_<Company>.pdf

If you cannot create files directly, provide the full content in markdown format clearly labeled for each document.

Mandatory resume-context convention

## Resume Creation Workflow (MANDATORY)
Every time a resume is requested, follow this exact workflow:

1. **Create a company-named folder** under `Resumes/2026/<CompanyName>/`
2. **Generate Resume** in both `.docx` and `.pdf` formats
   - Naming: `Rajesh_Dammala_<CompanyName>_<RoleShortName>.docx` / `.pdf`
3. **Generate Cover Letter** in both `.docx` and `.pdf` formats
   - Naming: `Rajesh_Dammala_CoverLetter_<CompanyName>.docx` / `.pdf`
4. **Create Job_Details.md** inside the company folder containing:
   - Job link/URL
   - Company name
   - Position title
   - Location
   - Salary range (if available)
   - Full job description from the prompt
   - Application status tracker

How to recreate

  1. Open AI_Resume_Prompt.md and copy the reusable prompt.
  2. Load resume-context.md for career context, achievements, naming rules, and folder workflow.
  3. Paste the target job posting URL/text into a new AI session.
  4. Ask for the full bundle: company folder + resume (.docx/.pdf) + cover letter (.docx/.pdf) + job details markdown.
  5. Reuse existing folders like PureStorage, Amazon, CoreWeave, or Itron as structural references.
